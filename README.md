 ## :information_source: How To Use
 
 ```bash
 # Clone this repository
 $ git clone https://github.com/mesquini/gobarber

> cd gobarber
> use file .json in the [Insomnia](https://insomnia.rest/download/)
> cd backend
> yarn install
> yarn dev
> yarn queue
 ```
